# 反響管理ルール

## 収集項目

| 媒体 | 収集指標 |
|------|----------|
| note | スキ・コメント・PV |
| Qiita | LGTM・ストック・PV・コメント |
| Zenn | いいね・PV・コメント |
| WordPress | PV・滞在時間・直帰率・コメント |
| 社内Git | 閲覧数・コメント・リアクション |

## 収集タイミング

- 投稿後1週間: 初動の反響を記録
- 投稿後1ヶ月: 安定期の数値を記録

## 分析観点

- どの媒体が最も反応が良かったか
- どのセクション・トピックが刺さったか
- コメントから得られるインサイト
- 次回に活かせる改善点

## 記録先

- 各記事の track.md に記録
- 次回の strategy.md に「前回の学び」として反映
